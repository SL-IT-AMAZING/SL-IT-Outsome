## Design System Insights

### Common Elements Across All Pages

#### Typography
- Base font size: 14px (forms) to 16px (Tally)
- Clean, readable fonts
- Korean + English bilingual support

#### Input Styling Patterns
- Rounded corners (8-9px border-radius)
- Light, semi-transparent backgrounds
- Minimal borders or no borders
- Generous padding for touch-friendliness

#### Color Palette
- Light backgrounds with semi-transparent overlays
- Dark text for readability
- Black CTA buttons
- White/cream accent colors

#### Form Design Philosophy
- Clean, minimal aesthetic
- Clear visual hierarchy
- Required field indicators
- Privacy/legal notices included

### Platform Differences

1. **Tally Form (Apply Now)**
   - 8px border-radius
   - White semi-transparent (0.5 opacity)
   - 16px font size
   - 373px fixed width

2. **Webflow Forms (Request Demo)**
   - 9px border-radius
   - Dark semi-transparent (0.06 opacity)
   - 14px font size
   - Responsive widths (336px/684px)

3. **Notion Page (Programs)**
   - Standard Notion formatting
   - Database-driven content
   - Tabbed navigation
   - Collapsible sections

All four sub-pages have been successfully captured and analyzed:

1. **Apply Now** - Tally-hosted multi-page application form with Korean/English bilingual support
