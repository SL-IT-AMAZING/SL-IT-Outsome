**Common Errors**:

| 에러 | 원인 | 해결법 |
|------|------|--------|
| `command not found: claude` | Node.js 미설치 또는 PATH 미등록 | Node.js 재설치, 터미널 재시작 |
| `EACCES permission denied` | npm 글로벌 설치 권한 부족 | `sudo npm install -g` 또는 nvm 사용 |
| `Invalid API key` | API 키 오타 또는 미등록 | 키 재확인, `export ANTHROPIC_API_KEY=sk-...` |
| `address already in use :3000` | 다른 프로세스가 포트 점유 | `lsof -i :3000` → `kill` 또는 포트 변경 |
| WSL2 설치 실패 (Windows) | 가상화 미활성 | BIOS에서 가상화 활성화 가이드 |
